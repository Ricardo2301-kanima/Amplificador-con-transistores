# Amplificador con Transistores BJT

Proyecto universitario enfocado en el **diseño, simulación y construcción de un amplificador de audio basado en transistores BJT**, utilizado para alimentar un altavoz.

## Contenido del repositorio

El repositorio incluye los siguientes archivos relacionados con el proyecto:

- **BJT.pptx**  
  Presentación utilizada para exponer el proyecto.

- **Informe.pdf**  
  Documento con la explicación completa del diseño, cálculos, desarrollo y resultados del amplificador.

- **Simulado.txt**  
  Archivo con la simulación del circuito.  
  Para visualizarlo:
  1. Ir a https://www.falstad.com/circuit/circuitjs.html  
  2. Importar el archivo en el simulador.

- **codigo.m**  
  Script de **MATLAB** utilizado para calcular los valores iniciales de los componentes del circuito.

## Objetivo del proyecto

Diseñar y construir un **amplificador de audio basado en transistores BJT**, verificando su funcionamiento mediante simulación y cálculos teóricos antes de su implementación física.
